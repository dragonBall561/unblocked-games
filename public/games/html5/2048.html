2048

Credits to dawong8's 2048 on GitHub


Use the Arrow Keys to move the Tiles. Tiles with the same value can be combined if they are next to each other. After each turn, a new tile will spawn randomly. Move the tiles to earn a score of 2048.
However, You can score more than 2048, the game ends only when you are out of moves or decide to quit.

Out of moves. You lose.
